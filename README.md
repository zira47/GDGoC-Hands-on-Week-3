# GDGOC Week 3 - JavaScript Cashier

Program JavaScript yang mensimulasikan mesin kasir sederhana. Menggunakan array, fungsi, dan for loop untuk menampilkan daftar pesanan, menghitung total belanja, menambahkan PPN 11%, lalu mencetak struk di console.

## Features
- Menyimpan data pelanggan
- Menyimpan daftar menu dan harga menggunakan array
- Menggunakan `for loop` untuk memproses pesanan
- Menghitung total belanja otomatis
- Menambahkan PPN 11%
- Menampilkan struk kasir di console

## Technologies
- JavaScript
- Node.js
- Console Output
